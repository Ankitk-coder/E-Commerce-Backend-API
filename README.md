# 🛒 ShopSphere - E-Commerce Backend API

A production-ready E-Commerce Backend application built with **Java 17**
and **Spring Boot 3**.

## Features

-   JWT Authentication
-   User Registration & Login
-   Product & Category Management
-   Shopping Cart
-   Order Management
-   Wishlist
-   Swagger/OpenAPI
-   Global Exception Handling
-   Validation
-   Docker Ready

## Tech Stack

-   Java 17
-   Spring Boot 3
-   Spring Security
-   Spring Data JPA
-   Hibernate
-   MySQL
-   Maven
-   Swagger
-   Docker
-   JUnit 5

## Project Structure

``` text
src
├── controller
├── service
├── repository
├── entity
├── dto
├── security
├── config
└── exception
```

## Getting Started

``` bash
git clone https://github.com/your-username/shopsphere.git
cd shopsphere
mvn clean install
mvn spring-boot:run
```

## API Documentation

After running the application:

`http://localhost:8080/swagger-ui/index.html`
